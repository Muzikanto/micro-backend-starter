# Micro Services Demo

A sample repo for demoing a micro services architecture setup.

## Getting started

1. Run: `npm run dev`
2. Navigate to `http://localhost:4000/`

Main Host App: `http://localhost:4000/`
Microservice Remote: `http://localhost:4001/`
